## React Template

React Template Project using:

- ES6
- Node
- React
- React Router
- React Bootstrap
- React Flux
- Webpack
- Webpack Dev
- Express

Deployed on Heroku at https://still-escarpment-70352.herokuapp.com/

## Installation and Development

```
git:clone -a still-escarpment-70352
```

Development
```
npm start
```
Production
```
npm run build
```
